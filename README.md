# Computing Method

* 回代法
* 追赶法
* 列主元高斯消去法

## 目录结构

lib/util.js里面是实现细节，包含许多内部辅助函数

test/test.js里面测试了实现的算法，可以直接在里面更改测试数据来测试

bin/`[method].js`里面是对应算法的控制台版本给老师测试，因为读文件版本不方便实现（这个命令行版本还借用了第三方库才方便实现），所以只实现了这个。demo如下:

```
$ bin/backSubstitution.js
请输入矩阵U的行数和列数
3 3
请输入矩阵U
1 2 3
4 5 6
7 8 9
请输入矩阵y
10 11 12
[ 4.8, 0.6, 1.3333333333333333 ]
```
